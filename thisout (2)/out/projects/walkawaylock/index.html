Technický přehled

WalkAway-Lock automaticky zamkne váš laptop, pokud se přednastavené Bluetooth zařízení dostane mimo dosah. Pomocí knihovny Bleak detekuje přítomnost zařízení s latencí ~3 s. Lehký, snadno konfigurovatelný kód, ideální pro Hackathony nebo veřejná místa.

"Bezpečnost je tak silná, jako její nejslabší článek."

Instalace a spuštění

walkaway.py
# Install dependencies
pip install bleak asyncio subprocess time

# Configure your device
RSSI_THRESHOLD = -85
MAC = "XX:XX:XX:XX:XX:XX"

# Run the script
python walkaway.py
